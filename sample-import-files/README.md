# Smart ERP Sample Import Files

This folder contains ready-to-use showcase files for every current bulk import target in the ERP.

Available import targets:

- `admissions`
- `students`
- `staff`
- `fees`
- `courses`
- `placements`
- `announcements`

Each target includes:

- one `.csv` file
- one `.xlsx` file

Suggested demo order:

1. `students.csv` or `students.xlsx`
2. `staff.csv` or `staff.xlsx`
3. `courses.csv` or `courses.xlsx`
4. `admissions.csv` or `admissions.xlsx`
5. `fees.csv` or `fees.xlsx`
6. `placements.csv` or `placements.xlsx`
7. `announcements.csv` or `announcements.xlsx`

Notes:

- The values are sample demo data, not production data.
- Some rows intentionally reference plausible related records so the files feel realistic in a live demo.
- If you want to showcase import preview, use the file first in preview mode before committing it.
